MeteoGuess : Divertir, Apprendre, Comprendre


Projet de deuxième année de BTS SIO

MeteoGuess est une application ludique développée dans le cadre de mon projet de deuxième année de BTS SIO. Son objectif principal est de permettre aux utilisateurs de tester leurs connaissances en devinant la température actuelle dans différentes villes du monde. En utilisant l'API OpenWeather, l'application fournit des données météorologiques en temps réel, offrant ainsi une expérience interactive et éducative.

  • Intégration d'API
  • Manipulation d'API
  • Conception d'UI
  • Gestion des requêtes réseau

Compétences mises en oeuvre

  • Programmation Android : Utilisation d'Android Studio pour concevoir une interface utilisateur intuitive et réactive.

  • Intégration d'API : Intégration de l'API OpenWeather pour récupérer les données météorologiques actualisées.

  • Traitement des données JSON : Extraction et manipulation des données JSON fournies par l'API OpenWeather.

  • Gestion de l'Interface Utilisateur (UI) : Création de vues dynamiques pour présenter les questions du quiz et afficher les résultats.



Fonctionnalités Principales

  • Quiz Météo : Les utilisateurs peuvent tester leurs connaissances en essayant de deviner la température actuelle dans une ville donnée. Ils ont le choix entre trois options et peuvent voir immédiatement la réponse correcte.

  • Prévisions Météo : En plus du quiz, les utilisateurs peuvent consulter les prévisions météorologiques pour les prochains jours dans la ville sélectionnée.

  • Recherche de Ville : Un champ de recherche permet aux utilisateurs de changer de ville et de découvrir les conditions météorologiques dans différentes régions du monde.



Pour résumer, MeteoGuess est un projet intéressant pour se familiariser avec la manipulation d'API, tout en créant une expérience divertissante et en permettant de mettre les utilisateurs en compétition au travers d'un simple quiz qui saura intéresser les plus curieux.
